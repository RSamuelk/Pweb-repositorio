3. Glossário Técnico

HTML
Linguagem de marcação utilizada para estruturar páginas web
CSS
Linguagem responsável pela estilização de elementos na página
JavaScript
Linguagem de programação voltada para interatividade
Também pode ser utilizada no desenvolvimento back-end
DOM
Estrutura que representa os elementos de uma página web
Permite a manipulação dinâmica do conteúdo
API
Interface que permite a comunicação entre sistemas
Conjunto de regras para integração de serviços
HTTP
Protocolo de transferência de dados na web
URL
Endereço utilizado para acessar recursos na internet
DNS
Sistema responsável por traduzir nomes de domínio em endereços IP
Front-end
Client-side
Parte da aplicação que interage diretamente com o usuário
Back-end
Server-side
Parte responsável pelo processamento e lógica da aplicação
