## Descrição
O layout usa CSS Grid para organizar a página principal e Flexbox para alinhar os blocos internos. A proposta é manter o HTML original sem alterações e criar um visual que faça referência a uma série que tenho muito apreço desde criânça, com suas cores variando entre tons de azul e branco, além de imagens com temas centrais do universo de Doctor Who.

## Recursos Utilizados
- CSS Grid
- Flexbox
- Cores
- Variáveis CSS
- Responsividade com media queries
- Imagens 
- Efeitos :hover
- Comentários organizando o código
